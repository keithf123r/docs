## Getting Started

### Prerequisites

Before setting up the Salesforce integration, ensure you have:

<CardGroup cols={2}>
<Card title="Salesforce Account Requirements" icon="computer">
A Salesforce organization with API access enabled
  - Administrator or System Integrator role
  - API access enabled in your Salesforce edition
</Card>

<Card title="Permission Setup" icon="lock">
  - Appropriate user permissions configured
  - API enabled user profile
  - Required object permissions (Lead, Contact, Account, Opportunity)
</Card>
</CardGroup>

### Quick Setup

<Steps>
  <Step title="Access Integration Settings">
    Navigate to Settings > Integrations > CRM in Alice
    <Frame>
    <img src="/platformCaptures/01-crm-provider-selection.png" alt="CRM Provider Selection" />
    </Frame>
  </Step>
  
  <Step title="Connect Salesforce">
    Click "Connect Salesforce" and authorize the connection
    <Frame>
    <img src="/platformCaptures/02-crm-authorization.png" alt="Salesforce Authorization" />
    </Frame>
  </Step>

  <Step title="Configure Sync Settings">
    Select the objects and fields you want to sync
    <Frame>
    <img src="/platformCaptures/03-field-mapping-configuration.png" alt="Field Mapping Configuration" />
    </Frame>
  </Step>
</Steps>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Salesforce disconnected">
    Verify your Salesforce connection is active
  </Accordion>
  
  <Accordion title="Leads are not deduplicating">
    Check your deduplication rules
  </Accordion>

  <Accordion title="Data is not pushed to correct objects or fields">
    Review field mappings and ensure they are correctly mapped
  </Accordion>

  <Accordion title="Data discrepancies">
    Check your synchronization configuration
  </Accordion>
</AccordionGroup>

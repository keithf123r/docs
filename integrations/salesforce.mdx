## Getting Started

### Prerequisites

Before setting up the Salesforce integration, ensure you have:

<CardGroup cols={2}>
<Card title="Salesforce Account Requirements" icon="computer">
A Salesforce organization with API access enabled
  - Administrator or System Integrator role
  - API access enabled in your Salesforce edition
</Card>

<Card title="Permission Setup" icon="lock">
  - Appropriate user permissions configured
  - API enabled user profile
  - Required object permissions (Lead, Contact, Account, Opportunity)
</Card>
</CardGroup>

### Quick Setup

<Steps>
  ### Access Integration Settings
  Navigate to Settings > Integrations in your Alice dashboard

### Connect Salesforce

Click "Connect Salesforce" and authorize the connection

### Configure Sync Settings

Select the objects and fields you want to sync

</Steps>
